# Verification & Identity

> Twyn uses multi-step verification including selfie liveness checks to ensure real people and prevent catfishing.

## How Verification Works

Twyn takes identity verification seriously to maintain a safe, authentic community.

### Age Verification
- Uses a **selfie liveness check** to verify users are real and of appropriate age
- Powered by DiDit, a third-party identity verification provider
- DiDit retains verification data for 30 days, then deletes it
- Twyn stores only the verification result (pass/fail), not the raw biometric data

### Optional ID Verification
- Users can optionally verify their identity with a government-issued ID
- Verified users receive a trust badge on their profile
- This is entirely optional — unverified users can still use all app features

### Optional Phone Verification
- Users can verify a phone number for additional account security
- This adds another layer of trust to the profile

## Authentication Methods

Twyn supports multiple ways to sign in:

- **Email**: Traditional email-based authentication
- **Phone OTP**: One-time passcode sent via SMS
- **Apple Sign-In**: Native Apple authentication
- **Google Sign-In**: Native Google authentication

## Privacy Commitments

- Verification data is processed securely and minimally
- Biometric data is not stored by Twyn — only the result
- Third-party verification providers are held to strict data retention policies
- Users can request deletion of all verification records

## Frequently Asked Questions

**Q: Do I need to verify my identity to use Twyn?**
A: Age verification via selfie is required. Full ID verification is optional but adds a trust badge to your profile.

**Q: What happens to my selfie after verification?**
A: The verification provider (DiDit) retains the data for 30 days then deletes it. Twyn only stores whether you passed or failed.

**Q: Can I sign in without Google or Apple?**
A: Yes. Twyn supports email and phone OTP authentication in addition to Apple and Google.
