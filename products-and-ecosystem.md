### ImpactAPI
An **open-source SDK** that estimates the environmental footprint of AI API calls. ImpactAPI provides approximate calculations for:

- **Energy consumption** per AI request
- **Carbon emissions** (CO₂e) per AI request
- **Water usage** per AI request

Calculations are region-aware and model-aware, using publicly available data sources to provide transparency about the environmental cost of AI usage.

- Website: [impactapi.co.uk](https://impactapi.co.uk)
- Open-source and free to use
- Designed for developers building AI-powered applications
